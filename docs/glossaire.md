# 📖 Glossaire PSI

| Terme | Définition |
|-------|------------|
| MEN | Ministère de l'Éducation Nationale du Togo |
| Bulletin | Document officiel trimestriel des résultats scolaires d'un élève |
| Tranche | Fraction de la scolarité annuelle (3 tranches généralement) |
| Coefficient | Pondération d'une matière dans le calcul de la moyenne générale |
| Rang | Position de l'élève dans le classement de sa classe (1er, 2ème...) |
| FCM | Firebase Cloud Messaging — service de notifications push Google |
| JWT | JSON Web Token — système d'authentification sans session serveur |
| DRF | Django REST Framework — extension de Django pour créer des API REST |
| SPA | Single Page Application — app web qui charge une seule page HTML |
| CRUD | Create Read Update Delete — les 4 opérations de base sur les données |
| API | Application Programming Interface — interface de communication entre systèmes |
| FCFA | Franc CFA — devise officielle du Togo |
| T1/T2/T3 | Trimestre 1 (Oct-Dec) / Trimestre 2 (Jan-Mar) / Trimestre 3 (Avr-Jun) |
| Kkiapay | Plateforme africaine d'intégration Mobile Money (MTN, Moov, Wave) |
| Africa's Talking | API SMS/USSD pour les télécoms africaines |
| VPS | Virtual Private Server — serveur cloud loué à l'heure/mois |
| Gunicorn | Serveur WSGI Python pour servir Django en production |
| Nginx | Serveur web proxy inverse pour exposer Django sur internet |
| WeasyPrint | Librairie Python qui convertit HTML/CSS en PDF |
| Flutter | SDK Google pour créer des apps iOS et Android depuis un seul code |
| Dart | Langage de programmation utilisé avec Flutter |
